Nuvio ClearKey — webOS

A webOS build of Nuvio that decrypts ClearKey DASH/HLS streams on-device (Shaka Player / EME) with no server-side proxy.

Install via Homebrew Channel

Add this repository URL in Homebrew Channel → Settings → Add repository:

https://retrocodes12.github.io/nuvioweb-clearkey/apps.json
  1. Install the Homebrew Channel on your LG TV (dev mode).
  2. Open it → Settings → Add repository → paste the URL above.
  3. Go back to the app list, find Nuvio ClearKey, and press Install.

Direct .ipk: releases/webos-v1.0.0 · Source: retrocodes12/nuvioweb-clearkey

Play a ClearKey stream by pointing a URL at manifest.mpd#clearkey=<kidHex>:<keyHex>.
